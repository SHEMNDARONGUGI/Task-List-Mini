# Task List Web App

A simple web application for managing and tracking tasks, built with HTML, CSS, and JavaScript.

## Features

- Add new tasks with a name, assignee, start date, and end date.
- Tasks are displayed in a table with their status ("Done" or "Not Done").
- Status is color-coded: green for "Done", red for "Not Done".
- User-friendly form validation and alerts for missing information.
- Responsive and visually appealing UI.

## Getting Started

1. **Clone or Download the Repository**

2. **Open `index.html` in your web browser**

   No build steps or dependencies are required.

## Project Structure

- `index.html`: Main HTML file for the app.
- `js/index.js`: JavaScript logic for adding and displaying tasks.
- `style/index.css`: Styles for the app.

## Usage

1. Enter the task details in the form:
   - Task name
   - Assigned to
   - Start date
   - End date (required to mark as "Done")
2. Click the **Add task** button.
3. The task will appear in the table below with its status.
